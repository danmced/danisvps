**Design Movement**: 8-bit Retro Gaming Aesthetic (1980s NES/Famicom era)

**Core Principles**:
- Strict pixel grid alignment with chunky, blocky forms
- Limited color palette reminiscent of early game consoles (16-32 colors max)
- Nostalgic charm through deliberate technical constraints
- Playful, game-like interactions that reward exploration

**Color Philosophy**: 
Drawing from classic 8-bit Valentine's Day games, we use a warm, limited palette: soft pinks (#FFB3D9, #FF6B9D), rich reds (#E63946, #C1121F), cream backgrounds (#FFF5E1), and dark outlines (#2B2D42). Colors are flat and saturated, never gradients. The emotional intent is nostalgic warmth mixed with playful innocence—like finding a love letter in an old Game Boy cartridge.

**Layout Paradigm**: 
Modular tile-based grid system where each section feels like a "screen" or "level" from a retro game. Asymmetric placement of UI elements mimics sprite positioning in classic games. Hearts, envelopes, and decorative elements snap to an 8px grid. Navigation uses pixelated arrows and game-style menus.

**Signature Elements**:
- Pixel-perfect heart sprites that pulse with frame-by-frame animation
- Chunky pixel borders (4-8px) around all containers and buttons
- Retro scanline overlay or subtle CRT screen curvature effect

**Interaction Philosophy**:
Every interaction should feel like pressing a button on a game controller—immediate, tactile, satisfying. Buttons have a "pressed" state with pixel-shift. Hover states show pixel sparkles or color swaps. Letter submission triggers a "level complete" style animation with pixel confetti.

**Animation**:
All animations use step-based timing (no easing curves) to maintain the digital, frame-by-frame aesthetic. Hearts bounce in 4-frame cycles. Text appears character-by-character like dialogue boxes in RPGs. Page transitions use pixel wipe effects (left-to-right or top-to-bottom reveals).

**Typography System**:
- Primary: "Press Start 2P" (Google Fonts) for headings and UI labels—authentic pixel font
- Secondary: "Silkscreen" (Google Fonts) for body text—more readable pixel font
- Hierarchy through size and color rather than weight (pixel fonts have limited weights)
- All text anti-aliasing disabled for crisp pixel edges

**Design Movement**: Kawaii Pastel Sticker Culture (Japanese cute aesthetic)

**Core Principles**:
- Soft, rounded forms with gentle curves everywhere
- Layered sticker-like elements with white borders
- Maximum cuteness through simplified, chibi-style illustrations
- Abundant decorative elements (stars, sparkles, ribbons)

**Color Philosophy**:
Ultra-soft pastel palette inspired by kawaii stationery: baby pink (#FFD6E8), lavender (#E6CCFF), mint (#CCFFE6), peach (#FFCCB3), and sky blue (#CCE6FF). Background is cream (#FFFBF5) with occasional white. The intent is to create a gentle, dreamy atmosphere that feels like opening a beautifully decorated letter from a friend—comforting, sweet, and uplifting.

**Layout Paradigm**:
Scrapbook-style asymmetric composition where elements appear "stuck" onto the page at playful angles (2-5 degree rotations). Letters and UI components have torn-paper edges or sticker borders. Sections overlap slightly with drop shadows to create depth. Navigation feels like flipping through a decorated journal.

**Signature Elements**:
- Pixel art hearts, envelopes, and cupids with kawaii faces (big eyes, rosy cheeks)
- Decorative pixel borders made of tiny hearts, stars, and flowers
- Floating pixel sparkles and bubbles throughout the interface

**Interaction Philosophy**:
Interactions should feel gentle and rewarding, like peeling a sticker or pressing a soft button. Hover effects add subtle glow or bounce. Clicks trigger cute sound effects (optional). Letter writing feels like decorating your own stationery with digital stickers.

**Animation**:
Soft, bouncy animations with slight overshoot. Hearts gently float upward. Stickers "pop" into place with a scale animation. Page transitions use soft fades or slide-ins with elastic easing. Pixel elements animate frame-by-frame but with smooth timing.

**Typography System**:
- Primary: "Fredoka" (Google Fonts, 600 weight) for headings—rounded, friendly, playful
- Secondary: "Quicksand" (Google Fonts, 400-500 weight) for body text—soft and readable
- Accent: "Silkscreen" (Google Fonts) for pixel art labels and small UI text
- Mix smooth fonts with pixel fonts to create contrast between "sticker" elements and "digital" elements

**Design Movement**: Brutalist Pixel Art (Contemporary digital brutalism meets retro computing)

**Core Principles**:
- Raw, unpolished pixel aesthetics with intentional "glitches"
- High contrast, bold color blocking
- Functional minimalism—every pixel serves a purpose
- Embrace of digital artifacts and technical limitations as beauty

**Color Philosophy**:
Stark, punchy palette: electric pink (#FF006E), cyber red (#FB5607), black (#000000), white (#FFFFFF), and neon yellow (#FFBE0B). No pastels, no gradients—only pure, saturated colors. The emotional intent is bold confidence and modern edge, like a digital love letter from the future. High contrast ensures accessibility while making a strong visual statement.

**Layout Paradigm**:
Grid-based but deliberately broken—sections align to a strict 16px grid, but elements intentionally overflow or misalign to create tension. Large blocks of solid color divide the page. Negative space is aggressive and purposeful. Navigation is stark and direct—no decoration, only function.

**Signature Elements**:
- Thick black pixel outlines (8-12px) around all major elements
- Dithered pixel patterns for texture and depth
- Glitch effects on hover (pixel displacement, color channel shift)

**Interaction Philosophy**:
Interactions are immediate and unforgiving—no hand-holding. Buttons change state instantly with no transition. Hover effects are dramatic (color inversions, glitch distortions). Letter submission is direct with a satisfying "data sent" confirmation. The experience feels powerful and intentional.

**Animation**:
Minimal, purposeful animations. No easing—only linear or step-based timing. Glitch effects use rapid frame changes. Text appears instantly or with a typewriter effect. Page transitions are hard cuts or pixel wipes. Movement is robotic and precise.

**Typography System**:
- Primary: "VT323" (Google Fonts) for all text—monospaced terminal aesthetic
- All text is single weight, hierarchy through size and color only
- Letter spacing increased for readability
- Occasional text glitch effects (color shift, duplicate characters)
</text>
